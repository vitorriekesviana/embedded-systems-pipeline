Project Overview

This project demonstrates a complete, end-to-end data pipeline, from hardware to a graphical user interface. I programmed an ATmega328P microcontroller in C++ to acquire temperature, humidity, and pressure data from a BME280 sensor via the I²C protocol. The data is then transmitted to a computer, where a Python script applies calibration parameters and displays the information in a real-time monitoring GUI.

Data Pipeline Stages

1

Data Acquisition

A microcontroller (C++) reads data from the BME280 sensor over I²C.

2

Data Transformation

A Python script receives the raw data and applies calibration formulas.

3

Real-Time Visualization

The processed data is displayed live on a custom-built Python GUI.
